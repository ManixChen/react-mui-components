# 快速启动指南

## 前置条件

1. 确保所有包都已构建
2. Node.js >= 16
3. npm >= 8

## 启动步骤

### 1. 构建所有包

在项目根目录运行：

```bash
cd ../..
npm run build
```

### 2. 安装依赖

```bash
cd examples/admin-template
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

## 功能说明

### 首页
- 展示所有可用组件
- 快速导航到各个示例

### 操作手册
- 完整的安装说明
- 每个组件的使用指南
- 代码示例

### 示例页面
- **表单示例**: 展示各种表单字段类型
- **表格示例**: 展示数据表格功能
- **请求示例**: 展示 HTTP 请求工具
- **加载示例**: 展示加载组件
- **富文本示例**: 展示富文本编辑器

## 注意事项

1. 如果使用本地包路径（file:），需要先构建所有包
2. 如果包已发布到 npm，可以修改 package.json 使用 npm 版本
3. 确保所有 peerDependencies 已安装
